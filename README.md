# 🤖 AI Season Bootcamp 01

This repository contains all the code, notes, assignments, and projects completed during **AI Season Bootcamp 01**. The bootcamp is designed to take learners from the fundamentals of AI agents to building production-ready agentic AI systems.

Throughout six weeks, the curriculum covers the complete AI agent development lifecycle, including prompt engineering, Retrieval-Augmented Generation (RAG), vector databases, multimodal AI, LangChain, LangGraph, tool calling, conversational memory, knowledge graphs, voice agents, Model Context Protocol (MCP), and the OpenAI Agent SDK.

## 🎯 Learning Outcomes

By completing this bootcamp, you'll learn to:

* Build autonomous AI agents
* Design reliable prompts
* Develop document-aware RAG applications
* Integrate vector databases
* Create multimodal AI systems
* Build stateful agent workflows
* Connect agents with external APIs and tools
* Implement conversational and long-term memory
* Develop voice-enabled AI assistants
* Structure production-ready AI applications

This repository serves as both a learning resource and a reference for building modern AI agent applications using industry-standard tools and frameworks.
